Nesta página pode ver as entidades intelectuais cujo prazo para eliminação ou revisão já venceu. Se as entidades intelectuais estão prontas para eliminação é possivel associá-las a um novo auto de eliminação ou alterar o seu calendário de eliminação. Se as entidades intelectuais estão prontas para revisão, apenas é possivel alterar o seu calendário de eliminação. Além disso, tambem é possivel ver entidades intelectuais cujo cálculo do período de conservação tenha falhado.
