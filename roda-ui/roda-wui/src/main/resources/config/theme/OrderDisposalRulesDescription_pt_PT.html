Nesta página é possivel alterar a ordem das regras de eliminação. É nesta página que é definida a prioridade com que as regras de eliminação são aplicadas ao repositório. A primeira posição na tabela corresponde ao nível máximo de prioridade, enquanto que a última posição na tabela corresponde ao nível mínimo de prioridade. Quando uma entidade intelectual nao é abrangida por nenhuma regra de eliminação, nenhum calendário de eliminação é associado.
